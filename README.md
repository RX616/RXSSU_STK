# RXSSU_STK
RXSSU管理器的SIM卡工具包版
# 实现方式
简单地描述一下，就是设备的root权限通过一个端口映射到STK菜单里，然后在SIM卡工具包中就可以管理root权限了，这个是可以在RXSSU管理器APP中自由开启和关闭的。
# 如何使用？
随便找一张无信号的废旧SIM卡，写入STK固件，然后插入到设备即可使用，但只能插入到SIM卡1才能读取。
# 有卡贴吗
卡贴的话可能会量产一波。
# 有eSIM吗
eSIM是有的，如果您的设备支持eSIM或有eSIM的实体卡那么装载eSIM即可使用STK版的RXSSU。
# 功能？
STK主页面会显示root状态，主页面有超级用户列表、模块列表、安装模块、设置和修补镜像。
这里的安装模块仅支持输入下载直链进行下载安装，修补镜像也是仅支持输入下载直链进行下载并修补，后续会支持file://+文件路径的链接。
然后模块的执行和 Web 也是完美支持的
